# secure_c_demo

Huawei C 安全函数库使用Demo

## Demo Usage (使用方法)
1. Clone code (把代码clone到Linux系统下)

```bash
git clone https://github.com/codehubcloud/secure_c_demo.git
```

2. Compile code (编译代码)

```bash
cd secure_c_demo
cd build
cmake ..
make
```

3. Run code (运行代码)

```bash
./demo_project
```

## Dependencies (依赖)

1. CMake
2. GCC
